# Move-Mint Frontend UI

A Next.js frontend application for the Move-Mint NFT minting platform with dance analysis capabilities.

## Features

- Video upload and dance analysis
- NFT minting with custom licenses
- Wallet integration
- Real-time progress tracking
- Session persistence

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Web3 Integration
- Jest for testing
